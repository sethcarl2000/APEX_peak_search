# APEX_peak_search
Code to perform peak-search using Profile-Likelihood methods for the APEX experiment

## Methods 
The statistical techniques employed here are based primarily on the Profile-Likelihood tests outlined in the paper _Asymptotic Formulae for Profile Likelihood Tests of New Physics_ from Glen Cowan _et. al._ [arXiv:1007.1727](https://arxiv.org/abs/1007.1727)

Histograms are represented using the [ROOT library](root.cern.ch). Many mathematical functions are also used from the ROOT framework.
